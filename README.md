# Criptrografia

## Sobre:

O código possui uma criptografia indecifrável por padrões, onde ele repete um loop (for) com uma equação para alterar cada letra para um valor completamente diferente do anterior, garantindo uma falta de padrão decifrável. Após rodar o código ele irá expor no terminal o texto criptografado e o texto descriptografado novamente.
A matemática por trás da criptografia é: 1. Etapa: o código pela o número de uma lista chamada "KeySoma" onde fica armazenado cada CharCode das letras do texto. 2. Etapa: o número faz uma multiplicação com uma variável chamada: "Position" que é adicionada 1 número a cada repetição do loop (for).
